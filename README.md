# Reinforcement_Learning_CartPole
Just learning the basics of RL with my first project (Cart Pole). The goal is to tech RL model to hold a stick straight in 2D simulation with real stick physiscs

## Overview

This project was built as part of my journey into reinforcement learning before applying RL to more complex real-world problems such as intelligent irrigation control.

The agent learns an optimal policy by interacting with the CartPole environment, receiving rewards for keeping the pole balanced, and improving its behavior over thousands of training episodes.

## Features

- Reinforcement Learning implementation
- OpenAI Gym/Gymnasium CartPole environment
- Training and evaluation pipeline
- Reward tracking
- Model saving and loading
- Performance visualization

## What I Learned

- Markov Decision Processes (MDPs)
- States, actions, and rewards
- Exploration vs exploitation
- Policy optimization
- Training neural networks through interaction instead of labeled data
- Evaluating RL agents

## Technologies

- Python
- Gymnasium (OpenAI Gym)
- Stable-Baselines3 (PPO)
- PyTorch
- NumPy
- Matplotlib

## Why I Built This

Before working on my smart irrigation project, I wanted to understand reinforcement learning fundamentals. CartPole helped me learn how RL agents interact with environments, optimize policies, and generalize learned behavior.

These concepts are now being applied to my larger project, where an RL agent learns irrigation strategies using AquaCrop simulations.
